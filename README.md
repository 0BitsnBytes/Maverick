# Maverick
..
A tilt rotar VTOL rc plane inspired by the V22 Osprey.
Uses 2 servo motors to tranition from vertical flight mode to forward flight mode.
The servos also act as control surfaces allowing us to use a mixing code to turn the plane in various directions
Uses a Teensy 4.1 with a MPU 6050 Gyro to stabilize the plane using the same mixing code stabilizing itself by differentiating the motor speeds and the tilt of the servos. 
The wing uses a NACA 4412 airfoil and is 3D printed. Each wing weighs approximately 300g when printed with PETG. It is supported with a carbon fibre tube that runs throughout the wing of the plane. 
We used a PWM type reciever.
The code is a tweaked version of dRehm flight adding a code to stabilize the plane as well as a mixing code to transition into different flight modes.
The entire plane was designed by us on onshape and printed entirely with PETG.
